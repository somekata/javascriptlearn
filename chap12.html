第12章: オブジェクト指向プログラミング（OOP）

  1. クラスの定義

    2. 　JavaScriptでのオブジェクト指向プログラミング（OOP）の基本は、クラスを使ってオブジェクトを定義することです。`class`キーワードを使って新しいクラスを作成できます。 

    // クラスの定義
class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        console.log(`こんにちは、${this.name}です！`);
    }
}

// クラスからインスタンスを生成
const person1 = new Person("太郎", 30);
person1.greet();  // "こんにちは、太郎です！"

  2. クラスの継承

    3. 　クラスの継承を使って、既存のクラスの機能を再利用することができます。`extends`キーワードを使って、別のクラスを拡張します。以下は、`Person`クラスを継承して`Student`クラスを作る例です。 

    // クラスの継承
class Student extends Person {
    constructor(name, age, studentId) {
        super(name, age);  // 親クラスのコンストラクタを呼び出す
        this.studentId = studentId;
    }

    study() {
        console.log(`${this.name}は勉強中です`);
    }
}

const student1 = new Student("次郎", 20, "S12345");
student1.greet();  // "こんにちは、次郎です！"
student1.study();  // "次郎は勉強中です"
