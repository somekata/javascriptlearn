目次 コード例 エスケープ変換

付録1　HTMLのサンプル

1. HTMLの基本構造


<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>サンプルページ</title>
  </head>
  <body>
    <h1>HTMLの基本構造</h1>
    <p>このページは、HTMLの基本構造を説明しています。</p>
    <a href="index.html">ホームに戻る</a>
  </body>
</html>

2. HTML要素の例


<h1>見出し1</h1>
<p>これは段落の例です。</p>
<a href="https://example.com">リンクの例</a>
<ul>
  <li>リストの項目1</li>
  <li>リストの項目2</li>
  <li>リストの項目3</li>
</ul>

<!-- 順序付きリスト (ol) の例 -->
<ol>
  <li>順序付きリストの項目1</li>
  <li>順序付きリストの項目2</li>
  <li>順序付きリストの項目3</li>
</ol>

<!-- 定義リスト (dl) の例 -->
<dl>
  <dt>定義語1</dt>
  <dd>定義語1の説明</dd>
  <dt>定義語2</dt>
  <dd>定義語2の説明</dd>
</dl>

3. フォーム要素の例


<form action="/submit" method="post">

  <input type="text" id="name" name="name"><br><br>
  
  <label for="email">メール:</label>
  <input type="email" id="email" name="email"><br><br>
  
  <input type="submit" value="送信">
</form>
