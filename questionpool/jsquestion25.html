JavaScript Quiz

問題25 次のコードを実行するとコンソールに表示される値はどれか。 

let str = "JavaScript is awesome!";
let modifiedStr = str
  .split(" ")
  .map((word, index) => index % 2 === 0 ? word.toUpperCase() : word.toLowerCase())
  .join("-");
let result = modifiedStr.length * modifiedStr.indexOf("AWESOME");
console.log(result);





解説

正解: b. 308
このコードの処理内容は以下の通りです:

  1. **str.split(" ")**:
    入力された文字列 "JavaScript is awesome!" を空白文字で分割し、次の配列を生成します:
    ["JavaScript", "is", "awesome!"]
  2. **map()**:
    配列の各要素に対して関数を適用し、偶数インデックス（0, 2）では大文字化（toUpperCase()）、奇数インデックス（1）では小文字化（toLowerCase()）します。結果は次の配列です:
    ["JAVASCRIPT", "is", "AWESOME!"]
  3. **join("-")**:
    配列を "-" で結合し、1つの文字列を生成します:
    "JAVASCRIPT-is-AWESOME!"
  4. **modifiedStr.length**:
    文字列 "JAVASCRIPT-is-AWESOME!" の長さを計算します:
    22 （"JAVASCRIPT"は10文字、"-"は2文字、"is"は2文字、"AWESOME!"は8文字）
  5. **modifiedStr.indexOf("AWESOME")**:
    部分文字列 "AWESOME" の開始位置を検索します。
    文字列 "JAVASCRIPT-is-AWESOME!" における "AWESOME" のインデックスは 14 です。
  6. **最終計算**:
    modifiedStr.length * modifiedStr.indexOf("AWESOME") を計算します:
    22 * 14 = 308
