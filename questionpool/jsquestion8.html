JavaScript Quiz

問題8 以下のコードを実行すると、コンソールに表示される値はどれか。 

let a = 10;
let b = 5;
let c = a + b;

a = c - b;
b = a * 2;
c = a + b;

console.log(c);





解説

正解: a. 30
このコードの処理内容は以下の通りです:

  1. let a = 10;, let b = 5;, let c = a + b;: c = 15
  2. a = c - b;: a = 15 - 5 = 10
  3. b = a * 2;: b = 10 * 2 = 20
  4. c = a + b;: c = 10 + 20 = 30

最終的に console.log(c) の出力は 30 となります。
