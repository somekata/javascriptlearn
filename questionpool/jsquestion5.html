JavaScript Quiz

問題5 以下のコードを実行すると、コンソールに表示される値はどれか。 

let stack = [1, 2, 3];
stack.push(4);
stack.pop();
stack.push(5);
console.log(stack);





解説

正解: d. [1, 2, 3, 5]
このコードの処理内容は以下の通りです:

  1. 初期値: 配列 stack = [1, 2, 3]。
  2. stack.push(4): 配列の末尾に 4 を追加します。結果: [1, 2, 3, 4]。
  3. stack.pop(): 配列の末尾の要素（4）を削除します。結果: [1, 2, 3]。
  4. stack.push(5): 配列の末尾に 5 を追加します。結果: [1, 2, 3, 5]。
  5. console.log(stack): 配列全体を出力します。

問題レベル：中級

難易度：☆☆

領域：配列操作
