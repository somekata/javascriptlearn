JavaScript Quiz

問題16 以下のコードを実行すると、コンソールに表示される値はどれか。 

let count = 0;
let total = 0;
while (count < 4) {
  total += count;
  count++;
}
console.log(total);





解説

正解: b. 6
このコードの処理内容は以下の通りです:

  1. 初期値: count = 0, total = 0
  2. while 文の条件: count < 4（count が 4 より小さい間ループ継続）
  3. ループごとの計算:
    • 1回目: total += 0 → total = 0, count++ → count = 1
    • 2回目: total += 1 → total = 1, count++ → count = 2
    • 3回目: total += 2 → total = 3, count++ → count = 3
    • 4回目: total += 3 → total = 6, count++ → count = 4
  4. 条件が偽（count = 4）となった時点でループ終了。
  5. console.log(total): 最終的に total = 6 が出力されます。

問題レベル：中級

難易度：☆☆

領域：while 文の基礎
