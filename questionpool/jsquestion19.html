JavaScript Quiz

問題19 以下のコードを実行すると、コンソールに表示される値はどれか。 

let fruits = ["Apple", "Banana"];
let action = "add";
let newFruit = "Cherry";

if (action === "add") {
  fruits.push(newFruit);
} else if (action === "replace") {
  fruits[1] = newFruit;
} else {
  fruits = [];
}

let result;
switch (fruits.length) {
  case 0:
    result = "No fruits";
    break;
  case 1:
    result = fruits[0];
    break;
  case 2:
    result = fruits.join(" and ");
    break;
  default:
    result = fruits.join(", ").concat(" are available.");
}
console.log(result);





解説

正解: d. Apple, Banana, Cherry are available.
このコードの処理内容は以下の通りです:

  1. 初期値: fruits = ["Apple", "Banana"], action = "add", newFruit = "Cherry"
  2. if 文の処理:
    • action === "add" が真なので、fruits.push(newFruit) が実行されます。
    • 結果: fruits = ["Apple", "Banana", "Cherry"]
  3. switch 文の処理:
    • fruits.length = 3
    • case 0: と case 1: と case 2: はスキップされます。
    • default: が実行され、fruits.join(", ").concat(" are available.") が計算されます。
    • 結果: "Apple, Banana, Cherry are available."
  4. console.log(result): 最終的に上記の文字列が出力されます。
