JavaScript Quiz

問題17 以下のコードを実行すると、コンソールに表示される値はどれか。 

let count = 5;
let result = 0;
do {
  result += count;
  count--;
} while (count > 0);
console.log(result);





解説

正解: b. 15
このコードの処理内容は以下の通りです:

  1. 初期値: count = 5, result = 0
  2. do while 文の動作:
    • do 部分は条件に関わらず少なくとも一度実行されます。
    • 条件 count > 0 が真の場合、ループを継続します。
  3. ループごとの計算:
    • 1回目: result += 5 → result = 5, count-- → count = 4
    • 2回目: result += 4 → result = 9, count-- → count = 3
    • 3回目: result += 3 → result = 12, count-- → count = 2
    • 4回目: result += 2 → result = 14, count-- → count = 1
    • 5回目: result += 1 → result = 15, count-- → count = 0
  4. 条件が偽（count = 0）となった時点でループ終了。
  5. console.log(result): 最終的に result = 15 が出力されます。

問題レベル：初級
