JavaScript Quiz

問題13 以下のコードを実行すると、コンソールに表示される値はどれか。 

let a = 2;
let b = 4;
let result = `Result:${a++} + ${--b} = ${a + b}`;
console.log(result);





解説

正解: b. Result:2 + 3 = 6
このコードの処理内容は以下の通りです:

  1. 初期値: a = 2, b = 4
  2. ${a++}: 後置インクリメントなので、a の現在の値（2）が式に使用され、a = 3 に増加します。
  3. ${--b}: 前置デクリメントなので、b の値が 1 減少して 3 になり、その値が式に使用されます。
  4. ${a + b}: a = 3, b = 3 なので、a + b = 6 です。
  5. 出力は次のようになります: 
    Result:
2 + 3 = 6

最終的に console.log(result) の出力は上記の通りとなります。
