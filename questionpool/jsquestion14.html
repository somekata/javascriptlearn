JavaScript Quiz

問題14 以下のコードを実行すると、コンソールに表示される値はどれか。 

let a = 5;
let b = --a + a--;
console.log(a, b);





解説

正解: a. 3, 8
このコードの処理内容は以下の通りです:

  1. 初期値: a = 5
  2. --a: 前置デクリメントで、a が 1 減少し 4 になります。
  3. a--: 後置デクリメントで、現在の値（4）が式で使用され、その後 a = 3 になります。
  4. 計算: b = --a + a-- は次のように評価されます:
    • --a = 4
    • a-- = 4
    • b = 4 + 4 = 8
  5. console.log(a, b): 最終的に a = 3, b = 8 となります。

最終的に console.log(a, b) の出力は 3, 8 です。

問題レベル：中級

難易度：☆☆

領域：デクリメント、演算子の優先順位
