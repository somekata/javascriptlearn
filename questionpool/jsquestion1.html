JavaScript Quiz

問題1 次のコードを実行するとコンソールに表示される値はどれか。 

let x = 5;
let y = 3;
let result = x + y;
console.log(result);




解説

正解: a. 8
このコードの処理内容は以下の通りです:

  1. let x = 5; と let y = 3;:
    変数 x に 5、y に 3 を代入します。
  2. let result = x + y;:
    x + y を計算し、その結果 8 を変数 result に代入します。
  3. console.log(result);:
    変数 result の値 8 をコンソールに表示します。

問題レベル：初級

難易度：☆

領域：基本的な数値演算
