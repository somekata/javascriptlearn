問題18 以下のコードを実行すると、コンソールに表示される値はどれか。 

let day = 3;
let dayName;
switch (day) {
  case 1:
    dayName = "Monday";
    break;
  case 2:
    dayName = "Tuesday";
    break;
  case 3:
    dayName = "Wednesday";
    break;
  case 4:
    dayName = "Thursday";
    break;
  case 5:
    dayName = "Friday";
    break;
  default:
    dayName = "Invalid day";
}
console.log(dayName);





解説

正解: c. Wednesday
このコードの処理内容は以下の通りです:

  1. 初期値: day = 3
  2. switch 文の動作:
    • case 1: 条件 day === 1 が偽なのでスキップ。
    • case 2: 条件 day === 2 が偽なのでスキップ。
    • case 3: 条件 day === 3 が真なので、このブロックが実行されます。
  3. dayName = "Wednesday" が設定され、break により switch 文が終了します。
  4. default 部分は実行されません。
  5. console.log(dayName): 最終的に "Wednesday" が出力されます。
