問題23 以下のコードを実行すると、コンソールに表示される値はどれか。 

function findSequence(target, query) {
    let matches = [];

    for (let i = 0; i <= target.length - query.length; i++) {
        let subsequence = target.slice(i, i + query.length);

        if (subsequence === query) {
            matches.push(i);
        }
    }

    return matches;
}

let targetSequence = "ATGCATGCGTACGATCG";
let querySequence = "ATGC";
let result = findSequence(targetSequence, querySequence);
console.log(`Result: ${result}`);





解説

正解: a. Result: 0,4
このコードの処理内容は以下の通りです:

  1. 関数 findSequence:
    • target: 対象となる塩基配列。
    • query: 検索する塩基配列。
    • matches: 部分一致が見つかった開始位置を格納する配列。
  2. ループ:
    • i は 0 から target.length - query.length まで増加。
    • slice(i, i + query.length) で現在の部分配列を取得。
    • 部分配列が query と一致した場合、matches に開始位置を追加。
  3. 対象配列:
    • targetSequence = "ATGCATGCGTACGATCG"
    • querySequence = "ATGC"
  4. 部分一致:
    • 0文字目から一致 → 開始位置 0。
    • 4文字目から一致 → 開始位置 4。
  5. 結果:
    • matches = [0, 4]。
    • console.log(result): Result: 0,4 を出力。
