JavaScript Quiz

問題12 以下のコードを実行すると、コンソールに表示される値はどれか。 

let a = 10;
if (++a > 10) {
  console.log(a);
} else {
  console.log(a - 1);
}





解説

正解: b. 11
このコードの処理内容は以下の通りです:

  1. 初期値: a = 10
  2. ++a: 前置インクリメントで a の値が 1 増加し、a = 11 となります。
  3. 条件式: ++a > 10 は 11 > 10 となり、条件は真（true）です。
  4. 条件が真なので、console.log(a) が実行され、a = 11 が出力されます。

もし条件が偽（false）であれば、console.log(a - 1) が実行されていたでしょう。

問題レベル：中級

難易度：☆☆

領域：条件式、前置インクリメント
