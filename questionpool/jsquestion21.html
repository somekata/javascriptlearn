JavaScript Quiz

問題21 以下のコードを実行すると、コンソールに表示される値はどれか。 

let result = "";
let counter = 1;

while (counter <= 5) {
  if (counter % 2 === 0) {
    result += `${counter * 2}-`;
  } else {
    result += `${counter * 3},`;
  }
  counter++;
}

result = result.slice(0, -1); // 最後の余計な文字を削除
console.log(result);





解説

正解: b. 3,4-9,8-15
このコードの処理内容は以下の通りです:

  1. 初期値: result = "", counter = 1
  2. while (counter <= 5): 条件が真の間、ループが実行されます。
  3. ループ内の条件分岐:
    • counter % 2 === 0: 偶数の場合、result に counter * 2 を追加し、区切り文字として "-" を追加します。
    • 奇数の場合、result に counter * 3 を追加し、区切り文字として "," を追加します。
  4. 各ループの処理:
    • 1回目 (counter = 1): result = "3,"
    • 2回目 (counter = 2): result = "3,4-"
    • 3回目 (counter = 3): result = "3,4-9,"
    • 4回目 (counter = 4): result = "3,4-9,8-"
    • 5回目 (counter = 5): result = "3,4-9,8-15,"
  5. result.slice(0, -1): 最後の余計な文字 ("," または "-") を削除します。
  6. console.log(result): 最終的な出力は "3,4-9,8-15" です。
