JavaScript Quiz

問題 次のコードを実行するとコンソールに表示される値はどれか。 

let score = 85;
let result;

if (score >= 90) {
  result = "A";
} else if (score >= 80) {
  result = "B";
} else if (score >= 70) {
  result = "C";
} else {
  result = "F";
}

console.log(result);




解説

正解: b. "B"
このコードの処理内容は以下の通りです:

  1. let score = 85;:
    score に値 85 を代入します。
  2. if (score >= 90):
    条件が成立しない（85は90以上ではない）ため、"A" は選ばれません。
  3. else if (score >= 80):
    条件が成立する（85は80以上）ため、result に"B" が代入されます。
  4. 残りの条件は評価されず、"B" がコンソールに表示されます。

重要ポイント:
条件分岐では、最初に成立した条件のみが実行され、残りの条件は無視されます。

難易度：☆

領域：条件分岐
