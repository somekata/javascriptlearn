JavaScript Quiz

問題4 以下のコードを実行すると、コンソールに表示される値はどれか。 

let x = 10;
function testScope() {
  let x = 20;
  if (true) {
    let x = 30;
    console.log(x);
  };
  console.log(x);
}
testScope();





解説

正解: b. 30, 20
このコードの処理内容は以下の通りです:

  1. グローバルスコープで let x = 10; が定義されますが、関数内では使用されません。
  2. 関数 testScope 内で新たに let x = 20; が定義され、関数スコープ内の x を上書きします。
  3. if ブロック内で再び let x = 30; が定義され、if のブロックスコープ内でのみ有効な x が作成されます。
  4. console.log(x)（if 内）はブロックスコープの x を参照し、30 を出力します。
  5. console.log(x)（if 外）は関数スコープの x を参照し、20 を出力します。

問題レベル：初級
