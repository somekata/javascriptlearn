JavaScript Quiz

問題24 次のコードを実行するとコンソールに表示される値はどれか。 

let numbers = [1, 2, 3, 4, 5];
let newNumbers = numbers.filter(num => num % 2 === 0);
console.log(newNumbers);





解説

正解: b. [2, 4]
このコードの処理内容は以下の通りです:

  1. let numbers = [1, 2, 3, 4, 5];
    配列 numbers を初期化します。この配列には 1 から 5 までの連続した数値が含まれています。
  2. numbers.filter(num => num % 2 === 0);
    filter メソッドは、配列の各要素に対してコールバック関数を適用し、条件を満たす要素だけを新しい配列として返します。
    この場合、条件 num % 2 === 0 は「偶数であること」を意味します。具体的には:
    • 1 % 2 === 1（条件不一致） → 除外される
    • 2 % 2 === 0（条件一致） → 新しい配列に含まれる
    • 3 % 2 === 1（条件不一致） → 除外される
    • 4 % 2 === 0（条件一致） → 新しい配列に含まれる
    • 5 % 2 === 1（条件不一致） → 除外される
    その結果、新しい配列 [2, 4] が生成されます。
  3. console.log(newNumbers);
    フィルタリングされた結果である新しい配列 [2, 4] がコンソールに出力されます。

重要ポイント:
filter メソッドは元の配列を変更しません。元の配列 numbers はそのまま保持され、新しい配列が返されます。
