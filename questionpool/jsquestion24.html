問題24 以下のコードを実行すると、コンソールに表示される値はどれか。 

function calculateMatchScore(seq1, seq2) {
  if (seq1.length !== seq2.length) {
    throw new Error("Sequences must be of equal length");
  }

  let score = 0;

  for (let i = 0; i < seq1.length; i++) {
    if (seq1[i] === "-" || seq2[i] === "-") {
      score -= 1;
    } else if (seq1[i] === seq2[i]) {
      score += 1;
    }
  }

  return score;
}

let seq1 = "ATG-CATG";
let seq2 = "ATGCAT-G";
let matchScore = calculateMatchScore(seq1, seq2);
console.log(`Match Score: ${matchScore}`);





解説

正解: c. Match Score: 2
このコードの処理内容は以下の通りです:

  1. アライメント:
    • seq1 = "ATG-CATG"
    • seq2 = "ATGCAT-G"
  2. 比較ルール:
    • 一致する塩基ペア: `+1点`
    • ギャップを含むペア: `-1点`
    • 不一致する塩基ペア: `0点`
  3. 各位置でのスコア:
    • 1文字目: AとA → +1点
    • 2文字目: TとT → +1点
    • 3文字目: GとG → +1点
    • 4文字目: -とC → -1点
    • 5文字目: CとA → 0点
    • 6文字目: AとT → 0点
    • 7文字目: Tと- → -1点
    • 8文字目: GとG → +1点
  4. 合計スコア: `+1 + 1 + 1 - 1 + 0 + 0 - 1 + 1 = 2`
