JavaScript Quiz

問題15 以下のコードを実行すると、コンソールに表示される値はどれか。 

let sum = 0;
for (let i = 1; i <= 5; i++) {
  sum += i;
}
console.log(sum);





解説

正解: b. 15
このコードの処理内容は以下の通りです:

  1. 初期値: sum = 0
  2. for ループの構造:
    • 初期化: let i = 1
    • 条件: i <= 5（i が 5 以下の間ループ継続）
    • 更新: i++（ループごとに i を 1 増加）
  3. ループごとの計算:
    • 1回目: sum += 1 → sum = 1
    • 2回目: sum += 2 → sum = 3
    • 3回目: sum += 3 → sum = 6
    • 4回目: sum += 4 → sum = 10
    • 5回目: sum += 5 → sum = 15
  4. 最終的に console.log(sum) の出力は 15 です。
