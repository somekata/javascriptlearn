JavaScript Quiz

問題10 以下のコードを実行すると、コンソールに表示される値はどれか。 

let b = 5;
let result = b++ + 3;
console.log(result);





解説

正解: b. 8
b++ は後置インクリメントで、現在の値が式に適用された後、b の値が増加します。

  1. 初期値: b = 5
  2. b++: 現在の値（5）が式で使用されます。
  3. 計算: result = 5 + 3 = 8
  4. b の値は式が完了した後に 1 増加し、b = 6 になります。

最終的に console.log(result) の出力は 8 となります。

問題レベル：初級

難易度：☆

領域：演算子、後置インクリメント
