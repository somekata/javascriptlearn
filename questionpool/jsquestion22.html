問題22 以下のコードを実行すると、コンソールに表示される値はどれか。 

function getComplementaryStrand(sequence) {
  let complement = "";
  for (let i = sequence.length - 1; i >= 0; i--) {
    let base = sequence[i].toUpperCase();
    switch (base) {
      case "A":
        complement += "T";
        break;
      case "T":
        complement += "A";
        break;
      case "C":
        complement += "G";
        break;
      case "G":
        complement += "C";
        break;
      default:
        throw new Error("Invalid nucleotide found: " + base);
    }
  }
  return complement.toLowerCase();
}

let dna = "aTgCaT";
let result = getComplementaryStrand(dna);
console.log(result);





解説

正解: c. atgcat
このコードの処理内容は以下の通りです:

  1. 初期値:
    • dna = "aTgCaT"
    • 塩基は大文字に変換されて処理されます。
  2. 各塩基に対して相補的な塩基を追加:
    • "A" → "T"
    • "T" → "A"
    • "G" → "C"
    • "C" → "G"
  3. 処理後:
    • 元の配列: "aTgCaT"
    • 順方向に処理され、相補鎖: "ATGCAT"
    • 最後に小文字に変換: "atgcat"
  4. console.log(result): 最終的に "atgcat" が出力されます。
