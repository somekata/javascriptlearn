第4章: JavaScriptの基本

　JavaScriptは、Webページに動的な要素を加えるためのスクリプト言語です。この章では、JavaScriptの基本的な要素を学びます。

  1. JavaScriptとは何か

    2. 　JavaScriptは、Webブラウザ上で動作するプログラミング言語です。ユーザーの操作に応じて、Webページの内容を動的に変更したり、アニメーションを作成したりすることができます。

  2. 変数とデータ型

    3. 　変数はデータを保存するための「箱」のようなものです。JavaScriptでは、`let`や`const`を使って変数を宣言します。また、データにはさまざまな型（数値、文字列、ブール値など）があります。第6章で詳しく説明します。 

    let name = "Alice";  // 文字列型
const age = 30;      // 数値型
let isStudent = true; // ブール型

  3. 演算子と計算

    4. 　JavaScriptでは、演算子を使って数値を計算したり、文字列を結合することができます。以下は基本的な演算子の例です。

    • 足し算: `+`
    • 引き算: `-`
    • 掛け算: `*`
    • 割り算: `/`
    let result = 10 + 5; // 結果は15

  4. 条件分岐（`if`文、`else`文）

    5. 　条件に応じて異なる処理を実行するために、条件分岐を使います。`if`文と`else`文を使用して、特定の条件が満たされた場合にのみコードを実行することができます。第8章で詳しく説明します。 

    let age = 20;
if (age >= 18) {
    console.log("成人です");
} else {
    console.log("未成年です");
}

  5. 繰り返し処理（`for`、`while`）

    6. 　繰り返し同じ処理を行いたい場合には、ループを使います。`for`ループや`while`ループを使って、指定回数または条件が満たされるまで処理を繰り返します。こちらも第8章で詳しく説明します。 

    // forループの例
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// whileループの例
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

  6. 関数の基本（関数宣言、関数式、アロー関数）

    7. 　関数は、複数の命令をまとめて実行できる便利な仕組みです。関数を定義しておけば、必要なときに何度でも呼び出すことができます。第9章で詳しく説明します。 

    // 関数宣言の例
function greet() {
    console.log("こんにちは");
}

// 関数式の例
const sayHello = function() {
    console.log("Hello");
}

// アロー関数の例
const add = (a, b) => {
    return a + b;
};

もっと知りたい人のコーナー: 演算子の応用

　ここでは、JavaScriptの演算子の中でもよく使われるものや、少し応用的な演算子について紹介します。

  1. 複合代入演算子 `+=`, `-=`, `*=`, `/=`

    2. 　複合代入演算子は、変数に対して演算を行った結果を、その変数に再度代入するために使います。以下は`+=`を使った例です。 

    let score = 10;
score += 5;  // scoreは15になります

    他の演算子も同様に動作します：

    • `-=`: 引き算して結果を変数に代入する
    • `*=`: 掛け算して結果を変数に代入する
    • `/=`: 割り算して結果を変数に代入する

  2. インクリメント `++` と デクリメント `--`

    3. 　`++`と`--`は、数値を1ずつ増やしたり減らしたりするための演算子です。 

    let count = 0;
count++;  // countは1になります
count--;  // countは再び0になります

    　`++`や`--`は、変数の前に置くか後に置くかで挙動が異なります。前置の場合は、まずインクリメントまたはデクリメントが行われ、その結果が返されます。後置の場合は、元の値が返された後にインクリメントやデクリメントが行われます。

  3. べき乗演算子 `**`

    4. 　JavaScriptでは、`**`を使って数値のべき乗を計算できます。例えば、2の3乗は以下のように計算できます。 

    let result = 2 ** 3;  // resultは8になります

  4. その他の演算子

    5. 　以下のような演算子もよく使われます：

    • `%`: 剰余演算子。割り算の余りを返します。例: `10 % 3`は`1`を返します。
    • `==`と`===`: 等価演算子。`==`は型の違いを無視して比較し、`===`は型も含めて厳密に比較します。
    • `!`: 否定演算子。値が`true`であれば`false`に、`false`であれば`true`に反転します。
