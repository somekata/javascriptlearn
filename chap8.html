第8章: 条件分岐と繰り返し処理

　JavaScriptでは、条件に応じて異なる処理を行うために「条件分岐」を使用します。また、特定の処理を繰り返すために「繰り返し処理（ループ）」を使用します。この章では、条件分岐と繰り返し処理の基本を学びます。

  1. 条件分岐（if、else if、else）

    2. 　条件分岐は、特定の条件が真（true）であるかどうかに基づいてコードの流れを制御します。最も一般的な方法は、`if`文を使った条件分岐です。さらに複雑な条件には、`else if`や`else`を組み合わせて使用します。 

    // if文の例
let age = 20;

if (age >= 18) {
    console.log("成人です");
} else if (age >= 13) {
    console.log("ティーンエイジャーです");
} else {
    console.log("子供です");
}

    条件式に基づく処理

    　`if`文の中で使われる条件は、比較演算子（`==`, `===`, `!=`, `!==`, `>`など）を使って書かれます。条件が`true`の場合、そのブロックのコードが実行されます。複数の条件を使いたい場合は、論理演算子（`&&`, `||`, `!`）を使って条件を組み合わせることができます。 

    // 複数条件の使用
let score = 85;
if (score >= 90) {
    console.log("優秀です");
} else if (score >= 70 && score < 90) {
    console.log("良い成績です");
} else {
    console.log("もっと頑張りましょう");
}

  2. switch-case構文

    3. `switch-case`構文は、特定の値に基づいて異なる処理を実行したい場合に使います。この構文は、複数の`if-else`文を簡潔にまとめるのに適しています。 

    // switch-caseの例
let day = 3;

switch (day) {
    case 1:
        console.log("月曜日");
        break;
    case 2:
        console.log("火曜日");
        break;
    case 3:
        console.log("水曜日");
        break;
    default:
        console.log("不明な曜日");
}

    if文で書いた場合

    `switch-case`構文は簡潔に書ける一方、同じ処理を`if-else`文で書くことも可能です。以下は、同じロジックを`if-else`文で書いた例です。 

    // if文で書いた場合の例
let day = 3;

if (day === 1) {
    console.log("月曜日");
} else if (day === 2) {
    console.log("火曜日");
} else if (day === 3) {
    console.log("水曜日");
} else {
    console.log("不明な曜日");
}

  3. 三項演算子

    4. 三項演算子（条件演算子）は、`if-else`文を1行で書ける簡潔な構文です。形式は`条件式 ? 真の場合の値 : 偽の場合の値`です。 

    // 三項演算子の例
let age = 20;
let status = (age >= 18) ? "成人" : "未成年";
console.log(status); // 成人

    if文で書いた場合

    三項演算子の簡潔な書き方に対して、同じ処理を`if-else`文で記述することもできます。以下は、その例です。 

    // if文で書いた場合の例
let age = 20;
let status;

if (age >= 18) {
    status = "成人";
} else {
    status = "未成年";
}

console.log(status); // 成人

    ネストされた三項演算子の例

    三項演算子は簡潔に条件分岐を書けますが、複数の条件がある場合にはネストすることもできます。以下は、年齢に基づいて「未成年」「成人」「高齢者」を分類する例です。 

    // ネストされた三項演算子の例
let age = 70;
let category = (age < 18) 
    ? "未成年" 
    : (age < 65) 
        ? "成人" 
        : "高齢者";

console.log(category); // 高齢者

    if文で書いた場合

    ネストされた三項演算子と同じ処理を、`if-else`文で書くこともできます。以下は、その例です。 

    // if文で書いた場合の例
let age = 70;
let category;

if (age < 18) {
    category = "未成年";
} else if (age < 65) {
    category = "成人";
} else {
    category = "高齢者";
}

console.log(category); // 高齢者

  4. 論理演算子を利用した条件分岐

    5. 条件に応じて値を返すときに論理演算子を使う方法です。

    AND (&&) 演算子

    条件が真の場合にのみ評価を実行します。

    // AND (&&) 演算子の例
const loggedIn = true;
const message = loggedIn && "Welcome, User!";
console.log(message); // "Welcome, User!"

    OR (||) 演算子

    条件が偽の場合にデフォルト値を設定します。

    // OR (||) 演算子の例
const input = null;
const result = input || "Default value";
console.log(result); // "Default value"

    Nullish Coalescing Operator (??)

    `null` または `undefined` の場合にデフォルト値を設定します。

    // Nullish Coalescing Operatorの例
const input = 0; // falsy value but not null or undefined
const result = input ?? "Default value";
console.log(result); // 0

  5. 繰り返し処理（for、while、do...while）

    6. 　繰り返し処理（ループ）は、同じ処理を複数回実行したい場合に使います。JavaScriptでは主に`for`、`while`、`do...while`ループが使われます。

    forループ

    　`for`ループは、決まった回数だけ繰り返すときに使われます。以下は、1から5までの数を出力する例です。 

    // forループの例
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

    whileループ

    　`while`ループは、条件が`true`である限り処理を繰り返します。条件が満たされなくなるとループが終了します。 

    // whileループの例
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

    do...whileループ

    　`do...while`ループは、少なくとも一度はループ内の処理を実行してから条件を確認します。最初に必ず一回処理が実行される点が、`while`ループとの違いです。 

    // do...whileループの例
let num = 0;
do {
    console.log(num);
    num++;
} while (num < 5);
