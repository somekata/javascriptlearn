第7章: 配列とオブジェクトの操作

　JavaScriptの配列とオブジェクトは、データを管理するための基本的なデータ構造です。この章では、それらの基本操作や、いつどのように使うべきかを学びます。

  1. 配列の基本操作

    2. 　配列は複数のデータを1つの変数に格納できるデータ構造です。配列の要素には、インデックスを使ってアクセスできます。以下の例では、配列を作成し、いくつかの操作を行います。 

    // 配列の宣言とアクセス
let fruits = ['りんご', 'バナナ', 'オレンジ'];
console.log(fruits[0]);  // 'りんご'が出力されます

    配列の操作（push、pop、shift、unshift）

    　配列に新しい要素を追加したり、既存の要素を削除する操作は非常に一般的です。以下は、代表的な配列操作メソッドの例です。 

    // 配列の操作
fruits.push('ぶどう');    // 'ぶどう'を追加
console.log(fruits);      // ['りんご', 'バナナ', 'オレンジ', 'ぶどう']

fruits.pop();             // 最後の要素を削除
console.log(fruits);      // ['りんご', 'バナナ', 'オレンジ']
    • push(): 配列の末尾に新しい要素を追加します。
    • pop(): 配列の末尾から要素を削除します。
    • shift(): 配列の最初の要素を削除します。
    • unshift(): 配列の最初に新しい要素を追加します。

    map()やfilter()の活用

    　配列を操作する際に、map()やfilter()といったメソッドを使うことで、要素を効率的に操作することができます。 

    // mapを使った例
let numbers = [1, 2, 3, 4];
let squaredNumbers = numbers.map(function(num) {
    return num * num;
});
console.log(squaredNumbers);  // [1, 4, 9, 16]

// filterを使った例
let evenNumbers = numbers.filter(function(num) {
    return num % 2 === 0;
});
console.log(evenNumbers);  // [2, 4]

  2. オブジェクトの基本操作

    3. 　オブジェクトは、キーと値のペアを持つデータ構造です。キーは一意で、値にはどんなデータ型でも格納できます。以下はオブジェクトの作成と基本的な操作の例です。 

    // オブジェクトの作成とアクセス
let person = {
    name: 'Alice',
    age: 25,
    city: 'Tokyo'
};

console.log(person.name);  // 'Alice'が出力されます

    オブジェクトの操作（追加、削除、更新）

    　オブジェクトのプロパティは簡単に追加、削除、更新することができます。以下にその例を示します。 

    // オブジェクトの操作
person.age = 26;             // プロパティの更新
person.country = 'Japan';    // 新しいプロパティの追加
delete person.city;          // プロパティの削除

console.log(person);  // { name: 'Alice', age: 26, country: 'Japan' }

  3. 配列とオブジェクトの使い分け

    4. 　配列は順序があるデータのリストを扱うのに適しており、オブジェクトは一意のキーを持つデータを扱うのに適しています。例えば、ユーザーの情報を管理する場合、各ユーザーをオブジェクトとして格納し、複数のユーザーを配列でまとめると良いでしょう。 

    // 配列とオブジェクトの組み合わせ
let users = [
    { name: 'Alice', age: 25 },
    { name: 'Bob', age: 30 },
    { name: 'Charlie', age: 35 }
];

console.log(users[1].name);  // 'Bob'が出力されます

参考：言語によって、配列の呼び名や仕組みがやや異なっています。参考までに、Pythonとの比較していますので、こちらをご覧ください。

オブジェクトのアクセス方法：ドット記法とブラケット記法

　オブジェクトのプロパティにアクセスするためには、ドット記法またはブラケット記法を使用します。次に例を示します。 

// ドット記法とブラケット記法
let users = { 
  name: 'Alice', 
  age: 25 
};

console.log(users.name);  // ドット記法によるnameプロパティへのアクセス　'Alice'が出力されます
console.log(users["name"]);  // ブラケット記法によるnameプロパティへのアクセス　'Alice'が出力されます

キーと値の取得

　オブジェクトのキーや値を取得するためには、Object.keys()やObject.values()を使用します。また、Object.entries()を使うと、キーと値のペアが返ります。 

let users = { 
name: 'Alice', 
age: 25,
nationality: 'Japan' 
};

cconsole.log(Object.keys(users));  // ['name', 'age', 'nationality']
console.log(Object.values(users));  // ['Alice', 25, 'Japan']
console.log(Object.entries(users));  // [['name', 'Alice'], ['age', 25], ['nationality', 'Japan']]

オブジェクトのスプレッド演算子

　スプレッド演算子を使うと、オブジェクトを簡単にクローンしたり、新しいプロパティを追加できます。 

const obj = { a: 1, b: 2 };

const newObj = {...obj, c: 3};  // { a: 1, b: 2, c: 3 }

オブジェクトの分割代入

　プロパティのアクセスを効率化するためには、分割代入を使うことができます。 

const { a, b, c } = { a: 1, b: 2, c: 3 };

console.log(a, b, c);  // 1, 2, 3
