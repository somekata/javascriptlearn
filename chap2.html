第2章: HTMLの基本

　HTML（HyperText Markup Language）は、Webページの基礎を作る言語です。この章では、HTMLの基本的な構造や要素について学んでいきます。

  1. HTMLタグの構造

    2. 　HTMLはタグで構成されており、各タグは要素を囲んでいます。HTML文書は<html>タグで始まり、<body>タグの中にページの内容が書かれます。 

    <html>
  <head>
    <title>ページのタイトル</title>
  </head>
  <body>
    <h1>見出し1</h1>
    <p>段落テキスト</p>
  </body>
</html>

  2. ブロック要素とインライン要素

    3. 　HTML要素には、ブロック要素とインライン要素があります。ブロック要素は、ページ全体で一つの大きな領域を占め、次の要素は新しい行から始まります。例えば、<div>や<p>タグはブロック要素です。一方、インライン要素は、要素の間に新しい行を作らずに他の要素と一緒に配置されます。<span>や<a>タグがインライン要素です。

    ブロック要素の例:

    <div>
  <h1>ブロック要素の見出し</h1>
  <p>これはブロック要素の段落です。</p>
</div>

    インライン要素の例:

    <p>
  このテキストには <span>インライン要素</span> が含まれています。
</p>

  3. 属性とプロパティ

    4. 　HTMLタグは属性を持つことができ、これにより要素に追加情報を与えて、見た目や動作を制御します。例えば、<a>タグにはhref属性があり、リンク先のURLを指定します。 

    <a href="https://example.com">こちらをクリック</a>

    　また、<input>タグにはvalue属性があり、要素の初期状態を指定します。 

    <input type="text" value="初期値">

    　一方、JavaScriptでは、HTMLの属性がDOMのプロパティとして扱われます。これにより、要素の状態を動的に操作することができます。 

    document.querySelector('input').value = "新しい値";
    　この場合、valueプロパティが"新しい値"に変更されます。

    HTMLとDOMの関係

    　HTMLの属性（valueなど）は、ページが読み込まれた後に、HTML要素がDOM（Document Object Model）に変換されます。その結果、DOMのプロパティとして扱われ、JavaScriptから操作可能になります。

    HTML属性とDOMプロパティの違い

    • HTML属性は、要素の初期状態を指定するために使用されます。例えば、<input>タグのvalue属性は、ページが読み込まれた時点での初期値を設定します。
    • DOMプロパティは、JavaScriptで操作可能な動的な値です。ページが読み込まれた後に、valueプロパティを変更することができます。

    <input type="text" value="初期値">
    • このHTMLでは、value属性は"初期値"として設定されています。ページが読み込まれた後、次のようにJavaScriptで変更可能です。
    document.querySelector('input').value = "新しい値";
    • ここで、valueプロパティが"新しい値"に変更されますが、HTMLの元のvalue属性は依然として"初期値"のままです。

  4. コメントの使い方

    5. 　HTML内では、コードの説明やメモを残すためにコメントを使うことができます。コメントはブラウザに表示されませんが、ソースコード上に残しておくことで、後から見直す際の助けになります。 

    <!-- これはコメントです -->
