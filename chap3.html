第3章: CSSの基本

　CSS（Cascading Style Sheets）は、Webページにスタイルを適用するための言語です。HTMLで作成したページのデザインやレイアウトを整え、視覚的に魅力的なページを作るために使用されます。この章では、CSSの基本的な書き方、セレクタ、プロパティ、そしてレイアウトの基礎を学んでいきます。

  1. CSSの基本的な書き方

    2. 　CSSは、HTML要素にスタイルを適用するために、セレクタとプロパティを使って書かれます。基本的な構文は以下の通りです。 

    セレクタ {
    プロパティ: 値;
}

    　例えば、以下のコードはすべての<h1>要素の色を青に設定します。 

    h1 {
    color: blue;
}

  2. 外部CSS、内部CSS、インラインCSS

    3. 　CSSをHTMLに適用する方法には、主に以下の3つがあります。

    • 外部CSS: 別ファイル（通常は.css拡張子）にスタイルを記述し、HTML内でリンクします（演習1）。
    • 内部CSS: HTMLファイルの<head>セクション内にスタイルを直接記述します。 
      <head>
    <style="color:#a0a0a0">
    p {
        color:#a0a0a0;
    }
    </style="color:#a0a0a0">
</head>
    • インラインCSS: HTML要素のstyle属性にスタイルを記述します。 
      <p style="color:#a0a0a0">

  3. セレクタとプロパティ

    4. 　セレクタは、どのHTML要素にスタイルを適用するかを指定する部分です。以下のセレクタの種類があります。

    • タグセレクタ: 特定のタグにスタイルを適用します。例: `h1 { color: red; }`
    • クラスセレクタ: クラス属性を持つ要素にスタイルを適用します。例: `.class-name { font-size: 20px; }`
    • IDセレクタ: ID属性を持つ要素にスタイルを適用します。例: `#id-name { margin: 10px; }`

  4. レイアウトとデザインの基礎（Flexbox、Grid）

    5. 　Webページのレイアウトには、FlexboxとCSS Gridがよく使用されます。これらの技術を使うことで、ページ内の要素を効率的に配置し、レスポンシブなデザインを実現することができます。

    Flexboxの例:

    .container {    
    display: flex;
    justify-content: center;
    align-items: center;
}

    Gridの例:

    .grid-container {    
    display: grid;
    grid-template-columns: 1fr 2fr;
    gap: 10px;
}

コラム2: 便利なツール

　必須ではないけれども、あると便利なツールをご紹介します。

  1. Visual Studio Code (VSCode)

    2. 　Visual Studio Code (VSCode) は、JavaScriptを学習する際に非常に便利なコードエディタです。無料で使えるうえに、HTML、CSS、JavaScriptの開発に必要な多くの機能を持っており、エクステンション（拡張機能）を追加することでさらに機能を拡張できます。以下の手順に従ってインストールしてください。

    インストール手順

    1. VSCodeの公式サイトにアクセスします。
    2. 自分の使用しているOS（Windows、macOS、Linux）に対応するインストーラを選択してダウンロードします。
    3. ダウンロードしたインストーラを開き、インストールを実行します。
    4. インストール後、VSCodeを起動し、初期設定を行います。
    5. 「拡張機能」タブから、HTML、CSS、JavaScript関連の拡張機能をインストールして、開発環境をさらに充実させましょう。

    便利な使い方

    　Enmetという自動入力機能が便利です。少し面倒なタグ付けを短縮することが可能です。例えば、 

    div.container>ul>li*5
    と入力してTabまたはEnterを押すと、 
    <div class="container">
    <ul>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
    </ul>
</div>
    と入力することができます。

    おすすめの拡張機能

    　以下の拡張機能をインストールすることで、コード入力やプログラム開発がよりスムーズになります。

    • Prettier - Code formatter: コードの自動フォーマットを行い、見やすく保つツールです。Windows/LinuxではShift + Alt + F、MacではShift + Option + Fでフォーマットします。また、設定で、Format On Saveの状態にしておくと、ファイルを保存するたびにPrettierが自動でコードをフォーマットしてくれます。
    • Live Server: HTMLファイルをリアルタイムでブラウザに反映し、変更を即座に確認できます。VSCodeの右下に「Go Live」というボタンが表示されます。このボタンをクリックすると、自動的にブラウザが起動し、ローカルサーバーでファイルがプレビューされます。
    • JavaScript (ES6) code snippets: JavaScriptの効率的なコーディングをサポートするコードスニペット集です。

  2. JSAnywhereアプリのインストール方法

    3. 　JSAnywhereは、モバイル端末でもJavaScriptコードを簡単に実行できる便利なアプリです。オンラインのコードエディタとしてだけでなく、アプリとしても利用可能です。以下は、iOSとAndroidのインストール方法です。

    iOS端末の場合（App Store）

    　iPhoneやiPadでJSAnywhereを使用するには、App Storeからインストールすることができます。以下の手順でアプリをダウンロードし、インストールを行ってください。

    1. iPhoneまたはiPadでApp Storeを開く。
    2. 検索バーに「JSAnywhere」と入力して検索。
    3. JSAnywhereアプリが表示されたら、「入手」ボタンをタップしてインストール。
    4. インストールが完了したら、アプリを開いてすぐにJavaScriptのコーディングを始められます。

    Android端末の場合（Google Play）

    　Android端末では、Google PlayからJSAnywhereアプリをインストールすることができます。以下の手順に従ってインストールを行ってください。

    1. AndroidスマートフォンやタブレットでGoogle Playを開く。
    2. 検索バーに「JSAnywhere」と入力して検索。
    3. 表示されたJSAnywhereアプリを選択し、「インストール」をタップ。
    4. インストールが完了したら、アプリを起動してすぐにJavaScriptの実行環境を使い始めることができます。

  3. Plicyへの登録と公開手順

    4. 　Plicyは、ゲームやアプリを公開できるプラットフォームで、JavaScriptを使って作成したプロジェクトを簡単に共有できます。自身の開発したコードを公開し、他のユーザーにプレイしてもらうことができます。登録や公開方法は以下の通りです。

    • Plicy公式サイトにアクセスし、無料アカウントを作成。
    • アカウント作成後、プロジェクトをzipファイルとしてアップロード。
    • 公開設定を行い、ゲームやアプリをPlicy上でシェアできます。

    　これにより、簡単に自身のJavaScript作品をオンラインで公開できます。

  4. GitHubのアカウント取得と公開手順

    5. 　GitHubは、ソースコードを保存・管理・共有できるプラットフォームです。以下の手順でアカウントを作成し、自分のプロジェクトを公開できます。

    • GitHub公式サイトにアクセスし、無料アカウントを作成。
    • アカウント作成後、リポジトリ（プロジェクトフォルダ）を作成。
    • 作成したリポジトリにソースコードをアップロード（GitコマンドまたはWebインターフェースを利用）。
    • リポジトリの公開設定を「Public」に変更し、他のユーザーとコードを共有可能。
    • GitHub Pagesを使用することで、Webサイトとしてプロジェクトを公開することも可能。

    　これにより、簡単に自身のコードやページをオンラインで公開し、オープンソースコミュニティに参加できます。

  5. GitBashのインストールと使用方法

    6. GitBashは、Git操作やコマンドライン作業を簡単に行えるWindows向けのツールです。以下の手順でインストールして使用します。

    インストール手順

    1. Git公式サイトにアクセスします。
    2. Windows用のGitをダウンロードしてインストールします。
    3. インストール中、デフォルトエディタやパス設定を確認します（推奨: Visual Studio Code）。
    4. インストール完了後、Git Bashを起動します。

    使用方法

    • Gitのバージョン確認: git --version を入力します。
    • JavaScriptの実行: Node.jsをインストールした後、node fileName.js で実行できます。

    以下のコードをGitBashで動かした際の表示例を示します。

    スクリプト例

    let arr1 = [1, 2, 3, 4];
let sliced = arr1.slice(1, 2);
console.log(`sliced:${sliced}`);
console.log("sliced:", sliced);

    このスクリプトを test.js として保存し、VScodeのTERMINALで以下のように実行します。

    ykaneko@YKanekoPC MINGW64 ~/<現在のディレクトリ>
$ node test.js

    すると、TERMINALには以下のように出力されます。

    sliced:2
sliced: [ 2 ]

  6. Google Colaboratoryの使用方法

    7. Google Colaboratory（Google Colab）は、Pythonコードをブラウザ上で実行できる無料のクラウド環境です。以下の手順で使用を開始できます。

    開始手順

    1. Google Colaboratoryにアクセスします。
    2. Googleアカウントでログインします。
    3. 新しいノートブックを作成し、Pythonコードを記述します。

    主な特徴

    • コードの実行: コードセルに print("Hello, Google Colab!") と記述し、再生ボタンをクリック。
    • ライブラリのインストール: !pip install ライブラリ名 で簡単にインストール可能。
    • データのアップロード: ローカルファイルをノートブックにアップロードして使用可能。

    Google Colabはデータ分析や機械学習に適しており、リアルタイムでのコード共有も可能です。 さらに、Google Colabは基本的にはPythonを動かすツールですが、Node.jsを使ってJavaScriptを実行することも可能です。

    以下のJavaScriptコードをGoogle Colabで動かした際の表示例を示します。

    スクリプト例

    let arr1 = [1, 2, 3, 4];
let sliced = arr1.slice(1, 2);
console.log(`sliced:${sliced}`);
console.log("sliced:", sliced);

    このスクリプトを test.js として保存する際の記述と操作方法を示します。

    左上の +コード をクリックして、コード入力画面を追加します。ここに以下を入力してください。

    %%writefile test.js
let arr1 = [1, 2, 3, 4];
let sliced = arr1.slice(1, 2);
console.log(`sliced:${sliced}`);
console.log("sliced:", sliced);

    左側の実行ボタンを押すと、以下のように表示されます。

    Writing test.js

    次に、もう一度 +コード をクリックして、以下を入力します。

    !node test.js

    再度、左側の実行ボタンを押すと以下のように表示されます。

    sliced:2
sliced: [ 2 ]
