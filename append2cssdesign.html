付録2　CSSのサンプル

1. レイアウト関連


- display：要素の表示方法を指定します。
    - block、inline、inline-block、flex、grid、none などが代表的です。
    - 例: display: block;
- position：要素の位置を指定します。
    - static、relative、absolute、fixed、sticky などがあります。
    - 例: position: absolute; top: 20px; left: 20px;
- float：要素を左右に浮かせるためのプロパティ。
    - left、right、none
    - 例: float: left;
- flex：フレックスボックスでレイアウトを作るために使います。
    - 例: display: flex; justify-content: center; align-items: center;
- grid：グリッドレイアウトを作るためのプロパティ。
    - 例: display: grid; grid-template-columns: 1fr 2fr;

2. サイズとスペース関連


- width / height：要素の幅や高さを指定します。
    - 例: width: 100%; height: 200px;
- max-width / max-height：最大の幅や高さを設定します。
    - 例: max-width: 100%;
- margin：外側の余白を指定します。margin: auto で中央寄せも可能。
    - 例: margin: 10px; または margin: 10px 20px;
- padding：内側の余白を指定します。
    - 例: padding: 20px;
- box-sizing：要素の幅や高さにパディングやボーダーを含めるかどうかを指定します。
    - content-box (デフォルト)、border-box
    - 例: box-sizing: border-box;

3. 背景と色の関連


- background-color：背景色を指定します。2つの記載方法があります。
    - 例: background-color: #f4f4f4;
    - 例: background-color: rgba(255, 255, 255, 0.8);
- background-image：背景画像を指定します。
    - 例: background-image: url('image.jpg');
- background-size：背景画像のサイズを指定します。
    - 例: background-size: cover;
- color：テキストの色を指定します。
    - 例: color: #333;
- opacity：透明度を指定します。
    - 例: opacity: 0.8;

4. テキスト関連


- font-size：フォントサイズを指定します。
    - 例: font-size: 16px;
- font-family：フォントの種類を指定します。
    - 例: font-family: Arial, sans-serif;
- font-weight：フォントの太さを指定します。
    - 例: font-weight: bold;
- line-height：行間の高さを指定します。
    - 例: line-height: 1.6;
- text-align：テキストの配置を指定します。
    - 例: text-align: center;
- text-decoration：テキストに下線や取り消し線を追加します。
    - 例: text-decoration: underline;
- letter-spacing / word-spacing：文字や単語の間隔を指定します。
    - 例: letter-spacing: 1px;

5. ボーダーやシャドウ


- border：要素のボーダーを設定します。
    - 例: border: 1px solid #ccc;
- border-radius：角を丸くします。
    - 例: border-radius: 5px;
- box-shadow：要素に影を付けます。
    - 例: box-shadow: 0 4px 8px rgba(0,0,0,0.2);

6. トランジション・アニメーション


- transition：要素が変化する際のトランジション効果を指定します。
    - 例: transition: all 0.3s ease-in-out;
- animation：要素にアニメーション効果を付けます。
    - 例: animation: fadeIn 2s ease-in;

7. レスポンシブデザイン


- @media：メディアクエリを使用して、画面サイズに応じたスタイルを適用します。
    - 例:
    @media (max-width: 600px) {
        body {
            background-color: lightblue;
        }
    }

8. その他の便利なプロパティ


- z-index：要素の重なり順を指定します。数値が大きいほど前面に表示されます。
    - 例: z-index: 100;
- visibility：要素を非表示にするが、レイアウトには残す。
    - 例: visibility: hidden;
- overflow：コンテンツが要素の領域を超えた場合の表示方法を指定します。
    - hidden、scroll、auto
    - 例: overflow: hidden;
