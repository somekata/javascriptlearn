第9章: 関数の基礎

　関数は、特定の処理をまとめて実行するための重要なツールです。JavaScriptの関数は、複数の命令を一つにまとめ、必要に応じて呼び出すことができます。この章では、関数の宣言方法や引数、戻り値について学びます。

  1. 関数の宣言と呼び出し

    2. 　JavaScriptでは、`function`キーワードを使って関数を宣言します。関数を呼び出すことで、その関数内に書かれたコードが実行されます。以下は、基本的な関数の宣言と呼び出しの例です。 

    // 関数の宣言
function greet() {
    console.log("こんにちは！");
}

// 関数の呼び出し
greet();  // "こんにちは！"が出力されます

  2. 引数と戻り値

    3. 　関数に引数を渡すことで、柔軟に処理を実行できます。引数は、関数を呼び出す際に値を渡すために使われ、関数内部でその値を使用します。また、関数は`return`文を使って結果を返すことができます。

    引数を持つ関数の例

    // 引数を持つ関数の宣言
function greetUser(name) {
    console.log("こんにちは、" + name + "さん！");
}

// 引数を渡して関数を呼び出す
greetUser("太郎");  // "こんにちは、太郎さん！"が出力されます

    戻り値を持つ関数の例

    // 戻り値を持つ関数の宣言
function add(a, b) {
    return a + b;
}

// 戻り値を受け取って使う
let sum = add(5, 3);
console.log(sum);  // 8が出力されます

  3. デフォルト値

    4. 　これはES6（ECMAScript 2015）で導入された機能です。デフォルト値を設定することで、引数が渡されなかった場合や undefined が渡された場合に、デフォルトの値を使用することができます。

    デフォルト値を設定するには、関数定義の際に引数に対して = を使って値を指定します。 

    function greet(name = "Guest") {
    console.log(`Hello, ${name}!`);
}

greet();                // 出力: Hello, Guest!
greet("Alice");         // 出力: Hello, Alice!
    • name 引数に値を渡さなかった場合、デフォルト値 "Guest" が使用されます。
    • 値を渡した場合、その値が優先されます。

  4. 関数式とアロー関数

    5. 　JavaScriptでは、関数を変数に格納することができ、これを「関数式」と呼びます。また、アロー関数（`=>`）を使って関数をより簡潔に書くこともできます。

    関数式の例

    // 関数式
const sayHello = function() {
    console.log("Hello!");
};

// 関数式の呼び出し
sayHello();  // "Hello!"が出力されます

    アロー関数の例

    // アロー関数
const multiply = (a, b) => a * b;

// アロー関数の呼び出し
let result = multiply(4, 5);
console.log(result);  // 20が出力されます

コラム4: 困った時のMDN

　プログラミングで困った時、特にWeb開発に関する質問がある場合には、Mozilla Developer Network（MDN）が頼りになります。MDNは、JavaScript、HTML、CSSなどのWeb技術に関する正確で信頼性の高い情報を提供しています。

　ただし、MDNの情報はプロフェッショナル向けであるため、説明がやや専門的なこともあります。しかし、詳細なリファレンスや正確な解説があるため、正しい情報を得たい時には非常に有用です。

おすすめの使い方

  • JavaScriptの関数やメソッドを確認する
  • HTMLタグやCSSプロパティの意味を調べる
  • ブラウザごとの互換性を確認する
